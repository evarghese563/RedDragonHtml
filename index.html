Red Dragon Archfiend

Red Dragon Archfiend is a Synchro card used by Jack Atlas in Yugioh. Red Dragon Archifend is currently not a meta deck as it's own but it is very good as a support for meta decks. Current meta decks that uses Red Dragon Archfiend is Centur-ion, Bystial, Albaz.
